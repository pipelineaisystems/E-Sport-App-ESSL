# Assets benötigt

Bitte erstelle folgende Bilddateien für die App:

## Erforderliche Assets

1. **icon.png** (1024x1024px)
   - App Icon für iOS und Android
   - Format: PNG mit Transparenz
   - Design: ESSL Logo oder Text "ESSL"

2. **splash.png** (1242x2436px für iOS, 1080x1920px für Android)
   - Splash Screen beim App-Start
   - Format: PNG
   - Hintergrund: Schwarz (#000000) mit ESSL Logo

3. **adaptive-icon.png** (1024x1024px)
   - Android Adaptive Icon
   - Format: PNG
   - Wird auf Android-Homescreen angezeigt

4. **favicon.png** (48x48px oder 192x192px)
   - Web Favicon
   - Format: PNG

5. **notification-icon.png** (96x96px)
   - Icon für Push Notifications
   - Format: PNG
   - Einfarbig (weiß) auf transparentem Hintergrund

6. **essl-bg.jpg** (optional, für Welcome Screen)
   - Hintergrundbild
   - Format: JPG
   - Empfohlene Größe: 1080x1920px oder größer

## Temporäre Lösung

Falls du die Assets noch nicht hast, kannst du:
1. Einfache Platzhalter-Bilder mit einem Bildbearbeitungsprogramm erstellen
2. Oder die app.json temporär anpassen, um ohne Assets zu starten
